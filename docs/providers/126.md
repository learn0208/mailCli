# 网易 126 邮箱配置指南

## 服务器参数

| 项目 | 值 |
|------|-----|
| Provider ID | `126` |
| 域名 | `126.com` |
| IMAP | `imap.126.com:993` |
| SMTP | `smtp.126.com:465` |

配置步骤与 163 相同：在网页邮箱开启 IMAP/SMTP，使用 **客户端授权密码** 作为 `MAILCLI_PASSWORD`。

- 示例配置：[examples/providers/126.yaml](../examples/providers/126.yaml)
- 详细说明可参考 [163.md](163.md)

```yaml
profiles:
  netease126:
    protocol: imap
    user: yourname@126.com
```
