# 网易 163 邮箱配置指南

## 服务器参数

| 项目 | 值 |
|------|-----|
| Provider ID | `163` |
| 域名 | `163.com` |
| IMAP | `imap.163.com:993`（SSL/TLS） |
| SMTP | `smtp.163.com:465`（SSL/TLS） |

## 开启 IMAP/SMTP 并获取客户端授权密码

1. 登录 [https://mail.163.com](https://mail.163.com)。
2. **设置** → **POP3/SMTP/IMAP**（或「客户端授权密码」相关入口）。
3. 开启 **IMAP/SMTP 服务**。
4. 按提示设置 **客户端授权密码**（又称授权码；与登录密码不同）。
5. `MAILCLI_PASSWORD` 填写该 **授权密码**，不要填网页登录密码。

## 配置文件

见 [examples/providers/163.yaml](../examples/providers/163.yaml)。

```yaml
profiles:
  netease163:
    protocol: imap
    provider: 163
    user: yourname@163.com
```

```bash
export MAILCLI_PASSWORD='客户端授权密码'
mailcli --profile netease163 search --limit 5
```

## 常见问题

- **535 / 认证失败**：几乎总是用了登录密码而非授权密码，或 IMAP 未开启。
- 企业邮、VIP 邮箱若域名不同，可能需手动填写 `imap.host` / `smtp.host`，并咨询网易帮助中心。
